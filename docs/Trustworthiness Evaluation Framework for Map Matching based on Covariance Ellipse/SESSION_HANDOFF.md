## 6. Key Design Decisions

1. **Paper positioning**: Systems/integration paper — novelty is the systematic integration of GNSS covariance+PL+integrity into HMM map matching, NOT any single algorithm component.
2. **ECE type**: Paper uses TW-based ECE (0.069), NOT EP-based ECE (0.262) or position-error ECE (0.121). Must be clear which is reported where.
3. **AUC defense**: CMM AUC=0.606 is lower than FMM AUC=0.965 because the low false-match rate (3.1%) leaves few discriminable wrong matches. FMM AUC is inflated by TW compression artifact (s.d. 0.039). Partial AUC and TW separation are better metrics.
4. **RAIM only**: Paper describes RAIM HPL (HPL = K·σ_major), NOT ARAIM. ARAIM is future work.
5. **Post-fix CMM**: All results use the 3% cumulative reverse guard fix (`cumulative_reverse_pct=0.03`). Pre-fix numbers (91.4% accuracy, 0.072 ECE) are documented in RECORDS.md for comparison.
